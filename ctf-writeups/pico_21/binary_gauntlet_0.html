Binary Gauntlet 0 Writeup

March 2021

Problem

This series of problems has to do with binary protections and how they affect exploiting a very simple program. How far can you make it in the gauntlet? gauntlet nc mercury.picoctf.net 12294

Solution

This challenge gives us a binary to download and develop the exploit for locally, and then an address to connect to and retrieve the flag from using the developed exploit.
When running the binary, it asks us for input, and then prints that input back out. Then it asks for input one more time.


In gdb we can see that the print is done using the printf function.



This makes it very likely that the binary is vulnerable to a format string exploit. We test this out using a few different parameters, and almost immediately get back the correct flag.
